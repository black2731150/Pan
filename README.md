# 这是一个云盘系统

## 项目结构
```
.
├── API.md          #api文档
├── config          #配置文件目录
├── config.yaml     #配置文件
├── dao             #数据处理层
├── global          #全局变量和自定义常量目录
├── logs            #日志文件
├── go.mod          #Go模块文件
├── go.sum          #Go依赖版本文件
├── initialize      #初始化相关代码目录\
├── models          #模板文件
├── pkg             #组件文件
├── main.go         #程序入口
├── README.md       #README文件
├── routers         #路由定义目录
├── services        #服务文件
├── storage         #用户上传的文件
├── utils           #公用工具代码
└── web             #前端项目

```
